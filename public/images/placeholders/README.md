## Image Guidelines

- **Hero Images**: Recommended size 1920x1080px (16:9 aspect ratio)
- **Content Cards**: Recommended size 350x200px (16:9 aspect ratio)
- **Format**: JPG or PNG
- **Optimization**: Compress images before uploading for better performance

## Notes

- If an image is missing, the system will automatically use a placeholder service
- Maintain consistent aspect ratios for better visual consistency
- Use descriptive file names that match the content
